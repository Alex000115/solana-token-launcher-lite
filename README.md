# Solana Token Launcher Lite

A streamlined, single-directory tool for launching SPL tokens on the Solana network. This repository provides a professional-grade script to handle the entire lifecycle of a token: creation, minting, and metadata initialization.

### Features
* **One-Click Creation:** Deploy a new SPL token to Devnet or Mainnet.
* **Minting Logic:** Easily mint a specific supply to your wallet.
* **Balance Checks:** Built-in utility to verify your SOL and token balances.
* **Professional Structure:** Clean, commented code designed for educational and production use.

### Quick Start
1. Install dependencies: `npm install`
2. Add your private key to `wallet.json`.
3. Run the launcher: `node launcher.js`

### Security Note
Never share your `wallet.json` or commit it to a public repository. This tool is for development purposes.
